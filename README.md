# Gantt Chart

## Getting Started

### Development

- `yarn install`
- `yarn run dev`

### E2E Tests

- `yarn run dev`
- `yarn run cypress:open` (in a separate terminal, for the time being)

### Linting

`yarn run lint`

### Formatting

`yarn run fmt:check` / `yarn run fmt:write`

### Building / Serving

- `yarn run build` 
- `yarn run start`
